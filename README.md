### Project Overview

This project simulates an Unmanned Ground Vehicle (UGV) navigating a battlefield environment. It demonstrates:

-> Shortest Path Calculation using Dijkstra’s Algorithm

-> Battlefield Grid Generation with random obstacles

-> Dynamic Obstacles Handling during navigation

-> The goal is to model how an intelligent agent (UGV) can navigate safely in a changing environment.

### Concepts Used

-> Graph representation (Adjacency Matrix)

-> Dijkstra’s Algorithm (Shortest Path)

-> Grid-based environment

-> Random obstacle generation

-> Dynamic environment simulation

### Project Structure
                     
--> Dijkstra.py  (Shortest path calculation)                   
--> Static_UGV.py  (Battlefield grid generation)                
--> Dynamic_UGV.py  (Dynamic obstacle simulation)                
--> README.md  (Documentation)

-> Python 3
